# notes-zhcn-portal

本仓库用于归档和发布一组独立的 HTML 页面。这些页面内容主要围绕技术笔记与信息整理，并以中文呈现。

## 项目简介

这是一个静态页面集合仓库。仓库中存放的 HTML 文件均为独立页面，不依赖特定域名或后端服务。每个页面均可直接通过浏览器打开，或在支持静态托管的平台上部署访问。

该仓库主要目的是集中保存和分享一些以 HTML 格式整理的内容，便于查阅、分发或存档。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 入口页面（可选）
├── pages/             # 存放主要 HTML 页面
│   ├── example.html   # 示例页面
│   └── ...
└── assets/            # 静态资源（CSS、图片、字体等）
    ├── style.css
    └── images/
```

- `pages/`：核心内容目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面引用的公共资源文件。

## 页面归档说明

- 所有页面均为纯前端 HTML，不涉及服务器端逻辑。
- 页面间无强依赖关系，可单独访问。
- 页面内容以中文编写，涵盖技术笔记、文档整理等主题。
- 如有更新，会直接在原文件上修改或新增文件，并记录于提交历史中。
